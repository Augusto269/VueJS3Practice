# VueJS3Practice
Git project to practice and learn VueJS3

Project Setup
Provide instructions on how to get your project up and running on a local development machine.

##Prerequisites
-Nodejs 18
-Npm 7.3.0
-Vue 3.0.0 or latest

Installation
-Clone the repository
-Install the dependencies -
```bash
npm install
```
-Run the project
```bash
npm run serve
```
